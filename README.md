# FraudShield AI
### Detect • Protect • Prevent

A simple system that helps users detect scams before losing money.

Team: Full Stack Attack  
Members: Aashwija, Charan K  

---

## Problem

First-time digital users are often targeted by scams such as:
- Fake lottery messages  
- OTP fraud  
- Fake payment links  
- Urgent money requests  

Most users realize the scam only after losing money.

---

## Solution

FraudShield AI analyzes messages, emails, and transactions in real time and:

- Detects the level of risk  
- Warns the user instantly  
- Explains why it is risky  
- Suggests what to do next  

---

## How It Works

User Input → Analysis Engine → Risk Score → Warning

---

## Features

- Risk Score (0–100)
- Message, Email, and Transaction Analysis
- Fake Link Detection
- Keyword Detection (OTP, urgent, lottery)
- Instant Scam Alerts
- Scam Explanation
- Guardian Assistant
- Simple Mode for easy understanding
- Dashboard with statistics and history
